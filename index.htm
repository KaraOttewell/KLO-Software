KLO Free Software Page

I recently got round to repairing and re-capping the power supply in my BBC Micro Model B, which I bought 40 years ago. Because I was downloading various things on my Windows PC so I wrote some Windows-based BBC Micro Utilities to display BBC Basic files, BBC BASIC data files, and disk images.
